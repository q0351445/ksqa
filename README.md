# 日常运维QA 搜索系统

从伙伴云三个数据源合并展示运维QA数据，支持关键字搜索。

## 功能

- 合并三个数据源（约5900条记录）
- 每页显示20条数据
- 支持关键字全文检索
- 响应式布局（PC/手机自适应）
- 定期自动更新数据

## 数据源

| 名称 | 记录数 |
|-----|--------|
| 日常运维QA | 27 |
| 日常运维QA（已满） | 5169 |
| 日常运维QA（记这个） | 704 |

## 使用

1. 本地运行: 直接打开 `index.html`
2. 更新数据: 运行 `python fetch_data.py`

## 定时更新

使用 cron 定时任务每天更新数据：

```bash
# 每天凌晨3点更新
0 3 * * * cd /path/to/qa_search && python fetch_data.py
```
