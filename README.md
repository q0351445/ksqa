# 运维知识库

从伙伴云三个数据源合并展示运维QA数据，支持关键字搜索。

## 功能

- 合并三个数据源（约5900条记录）
- 每页显示20条数据
- 支持关键字全文检索
- 响应式布局（PC/手机自适应）
- 图片支持（点击放大）

## 数据来源

| 名称 | 记录数 |
|-----|--------|
| 日常运维QA | ~27 |
| 日常运维QA（已满） | ~5169 |
| 日常运维QA（记这个） | ~719 |

## 访问地址

## 更新数据

在开发目录运行：
```bash
python fetch_data.py
```

然后同步并推送到GitHub：
```bash
sync_deploy.bat
push_to_github.bat
```
